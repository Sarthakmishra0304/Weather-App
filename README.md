# Weather-App
A responsive weather app that displays real-time weather data and multi-day forecasts using the OpenWeather API. Built with HTML, CSS, and vanilla JavaScript, featuring a clean glassmorphism UI and smooth horizontal scrolling.
A modern and responsive weather application that displays real-time weather information and a multi-day forecast for any city using the OpenWeather API. Built with vanilla JavaScript and a clean glassmorphism-inspired UI.

Features
Search weather by city name
Real-time temperature and weather conditions
Displays humidity and wind speed
Current date shown dynamically
Multi-day forecast with horizontal scrolling
Weather icons based on conditions
Responsive, modern glass-style UI

Tech Stack
HTML5
CSS
JavaScript 
OpenWeather API
Google Material Symbols

What I Learned
Fetching and handling data from external APIs
Using async/await for asynchronous operations
DOM manipulation and dynamic UI updates
Debugging real-world JavaScript and CSS issues
Handling flexbox overflow and horizontal scrolling
